Jeu du nombre mystere en html, css et javascript

Cet exercice à été réalisé en cours de développement en première année de BTS SIO.

Le but de l’exercice était d'écrire un programme en html, css et javascript

qui permettrait à l’utilisateur de deviner un nombre généré aléatoirement

dans une tranche choisis par le développeur en le saisissant dans une zone de saisie.

Ce programme devait :

Afficher un message indiquant si le nombre saisi par l’utilisateur est

trop petit ou trop grand par rapport au nombre mystère.

Afficher un message d’erreur si le nombre saisi est

supérieur au nombre maximal de la tranche.

Afficher un message d’erreur si des caractères sont saisis et non un nombre.

Compétences acquises :

- Ecrire des fonctions.

- Afficher des messages selon des conditions.

Cliquez sur l'image pour tester le jeu.
